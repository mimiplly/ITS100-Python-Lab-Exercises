# ITS100 – Introduction to Computers and Programming Lab Exercises

Welcome to my **ITS100 Introduction to Computers and Programming** lab exercises repository.

This repository contains a collection of Python programs and completed lab exercises from the course. It documents my progress and understanding of basic programming concepts throughout the semester.

---

## About the Course

ITS100 is an introductory programming course that provides a foundation in computer science and problem-solving.

The course covers topics such as:

- Basic programming concepts
- Variables and data types
- Conditional statements
- Loops
- Functions
- Lists and other data structures
- Algorithms and problem-solving
- Python programming

The lab exercises provide hands-on practice and help reinforce the concepts learned during lectures.

## Repository Structure

The repository is organized into separate folders or files for each lab exercise.
